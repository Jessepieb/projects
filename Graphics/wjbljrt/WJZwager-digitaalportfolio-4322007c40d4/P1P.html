P1P Race Game

Het eerste project van dit jaar was de race game. Het is natuurlijk leuk om te beginnen met een game maken. Ik heb dit project samen gewerkt met Jaimy Beima, Anne Pieter Boonstra, Frank Keuning, Robert Slomp en Jesse Piebenga. Bij dit project moesten we een race game maken in C# in een Windows Form App. We mochten dus geen libraries of frameworks gebruiken om ons te helpen. Dat was de grootste uitdaging.
